
# Оплата налога (по реквезитам)

## Шаг 1. На «Панели» появилась сумма налога к оплате

![Скриншот шага 1](https://taxflowge.github.io/images/pay_not_bog_step1.png)

После подачи декларации за период на экране **«Панель»** появляется карточка **«Оплата налога»** с рассчитанной суммой. Это означает, что декларация принята и теперь нужно оплатить налог за указанный период.

**Что нужно сделать:**

-  нажмите **«Оплатить налог»** на карточке **«Оплата налога»**.

**Результат шага:** откроется окно выбора способа оплаты.

**Примечание:** *на скриншоте скрыты чувствительные данные — у вас будут отображаться ваши реальные значения.*

---

## Шаг 2. Выбираем способ оплаты «Вручную»

![Скриншот шага 2](https://taxflowge.github.io/images/pay_not_bog_step2.png)

Откроется модальное окно **«Оплата налога за …»** с выбором способа оплаты. Для пользователей, которые оплачивают налог через другой банк (без автоматического создания платежа через BOG Business), используется вариант **«Вручную»** — оплата по реквизитам.

**Что нужно сделать:**

-  выберите **«Вручную»**.

**Результат шага:** откроется окно с реквизитами для оплаты.

**Примечание:** *на скриншоте скрыты чувствительные данные — у вас будут отображаться ваши реальные значения.*

---

## Шаг 3. Копируем реквизиты и оплачиваем в своём банке

![Скриншот шага 3](https://taxflowge.github.io/images/pay_not_bog_step3.png)

Откроется окно **«Оплата по реквизитам»**. Здесь показаны все данные, которые нужны, чтобы создать платеж в вашем банке (получатель, данные плательщика, код казначейства, назначение платежа и сумма). Рядом с полями есть кнопки копирования — можно быстро перенести реквизиты в банковское приложение/интернет-банк.

**Что нужно сделать:**

-  скопируйте реквизиты (кнопки копирования справа от полей);
-  создайте платеж в вашем банке и оплатите налог по этим реквизитам;
-  нажмите **«Закрыть»**.

**Результат шага:** вы оплатите налог через свой банк; после синхронизации с Revenue service статус оплаты обновится в TaxFlow автоматически.

**Примечание:** *на скриншоте скрыты чувствительные данные — у вас будут отображаться ваши реальные значения.*
